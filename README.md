# coding
Modify the file encoding format
   
##描述  
  
修改文件编码格式，例如：GBK转UTF8   
====================  
  
修改变量   
->
root_path : 需要进行转码的文件路径    
file_type : 需要进行转码的文件格式，比如html文件   
from_code : 文件的编码   
to_code   : 文件的目标编码   
   
    
##实例    

git clone https://github.com/baixuexiyang/coding.git    

查看html文件夹下的内容，会出现乱码    

执行node code.js   

再次查看就正常了
